객체지향 패러다임의 관점에서 핵심은 역할, 책임, 협력이다.
객체들이 애플리케이션의 기능을 구현하기 위해 수행하는 상호작용을 협력이라고한다.
협력에 참여하기 위해 수행하는 로직을 책임이라고 부른다.
책임들이 모여 객체가 수행하는 역할이라고한다.
